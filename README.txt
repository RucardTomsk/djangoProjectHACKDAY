Чтобы запустить наш проект:
1. Должен быть установлен python
2. Скачиваем ветку finish
3. Открываем папку HAKATON 30.12.2021, перенносим ее содержимое в *какую-то* папку
4. pip install jsonschema
   pip install ccxt
   pip install requests
   pip install eel
5. cd в папку, куда перенесли HAKATON
6. Запускаем из консоли mainWEB.py
